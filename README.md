# Selectable

Select code freely!

写代码的时候经常需要从网上“参考”一些最佳实践，
而某些“专业开发者社区”经常提示登录才能复制，
但是我个人就不喜欢在各种网站上保持登录状态，
因此每次需要“参考”的时候还得登录就很麻烦。

于是开发了这个Chrome插件，
需要“参考”的时候只要点一下icon就可以为所欲为了。

## 怎么用

- `git clone`
- 打开chrome 设置 更多工具 扩展程序
- 打开右上角开发者模式
- 点击左上角“加载已解压的扩展程序”
- 找到clone好的`selectable`文件夹，点击“选择文件夹”
- enjoy
